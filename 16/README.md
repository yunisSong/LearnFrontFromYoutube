`conic-gradient` 设置一个圆锥渐变

```css
background-image: conic-gradient(#fff 0deg, transparent 320deg);
```

![](demo1.png)

还可以设置多个值

```css
background: conic-gradient(
  deeppink 0,
  deeppink 30%,
  yellowgreen 30%,
  yellowgreen 70%,
  teal 70%,
  teal 100%
);
```
